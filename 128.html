#128 · Longest Consecutive Sequence
Given an unsorted array of integers nums, return the length of the longest consecutive elements sequence.

Key insight to recall

  • What data structure gives O(1) lookup?
  • How do you avoid counting sequences multiple times?
  • When should you start counting a new streak?
👆 Click card to reveal solution
✓ Solution HashSet
Algorithm steps
  • 1Put all numbers into a HashSet (deduplicates + O(1) lookup)
  • 2Loop each number — skip if num−1 exists (not a sequence start)
  • 3When it IS a start: walk forward with while (set.contains(cur+1))
  • 4Track longestStreak = max(longest, current)
Java skeleton
Set<Integer> set = new HashSet<>(); for (int n : nums) set.add(n); int longest = 0; for (int n : set) { if (!set.contains(n - 1)) { // sequence start int cur = n, streak = 1; while (set.contains(cur + 1)) { cur++; streak++; } longest = Math.max(longest, streak); } } return longest;
