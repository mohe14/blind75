Longest Palindromic Substring

Medium Expand From Center
Examples
"babad"→"bab" // "aba" also valid
"cbbd"→"bb"
"a"→"a"
💡 A palindrome mirrors around its center. Every center can be either a single char (odd) or a pair of chars (even). Try both for each index.
1 ≤ s.length ≤ 1000 lowercase English Return String
click to reveal solution

Solution — Expand From Center

Two Pointers
Odd Length
expand(s, i, i)
Same char as center — e.g. "aba"
Even Length
expand(s, i, i+1)
Pair as center — e.g. "abba"
Step-by-step
1Class-level: int start = 0, maxLen = 1 to track best window.
2Loop every index i. Call expand(s, i, i) and expand(s, i, i+1).
3In expand: while left ≥ 0, right ≤ s.length-1, and chars match → valid palindrome window.
4If window right - left + 1 > maxLen, update maxLen and start = left.
5Expand outward: left--, right++.
6Return s.substring(start, start + maxLen).
int start = 0, maxLen = 1;

String longestPalindrome(String s) {
  for (int i = 0; i < s.length(); i++) {
    expand(s, i, i);      // odd
    expand(s, i, i + 1); // even
  }
  return s.substring(start, start + maxLen);
}

private void expand(String s, int left, int right) {
  while (left >= 0 && right <= s.length()-1
         && s.charAt(left) == s.charAt(right)) {
    if (right - left + 1 > maxLen) {
      maxLen = right - left + 1;
      start = left;
    }
    left--; right++;
  }
}
Time: O(n²) Space: O(1)
