Serialize & Deserialize Binary Tree
Convert a binary tree into a string and reconstruct it back.

Think:
  • How do you preserve tree structure?
  • What traversal helps rebuild it?
  • How do you represent null nodes?

Solution · Preorder DFS + Queue
1. Use preorder traversal (root → left → right)
2. Append values to string
3. Use special marker "N" for null nodes
4. For deserialize:
  • Split string into queue
  • Rebuild tree recursively in preorder
String serialize(TreeNode root) { preorder(root); } TreeNode build(Queue q) { val = q.poll(); if (val == "N") return null; node = new TreeNode(val); node.left = build(q); node.right = build(q); }
Key Insight:
Preorder guarantees correct reconstruction because we always process root before its children. Null markers preserve structure.
Time: O(n) Space: O(n)
