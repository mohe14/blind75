#1 · Two Sum
Given an array nums and an integer target, return the indices of the two numbers that add up to target.

Each input has exactly one solution. You may not use the same element twice.

Key questions to ask yourself

  • What's the complement of nums[i] that you need to find?
  • What do you store in the map — value or index?
  • Do you check the map before or after inserting?
👆 Click card to reveal solution
✓ Solution HashMap<value → index>
Algorithm steps
  • 1Create HashMap<Integer, Integer> mapping value → index
  • 2For each nums[i], compute diff = target − nums[i]
  • 3Check map first — if diff exists, return {map.get(diff), i}
  • 4Otherwise, put nums[i] → i into the map and continue
Java skeleton
Map<Integer,Integer> map = new HashMap<>(); for (int i = 0; i < nums.length; i++) { int diff = target - nums[i]; if (map.containsKey(diff)) return new int[]{ map.get(diff), i }; map.put(nums[i], i); // store AFTER checking } return null;
Complexity
Time: O(n) Space: O(n)
