Solution · DP + Brian Kernighan's Trick
1
Init: create result[n+1]. result[0] = 0 implicitly — zero has no set bits, and serves as the DP base case.
2
DP recurrence: for each i, strip its lowest set bit to get a smaller index already computed, then add 1.
result[i] = result[i & (i−1)] + 1
i & (i−1) removes lowest set bit → already solved subproblem
