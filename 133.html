# 133 Clone Graph
Medium Graph DFS
Algorithm Steps
1
Define a class-level Map<Node,Node> called visited to track original → clone mappings.
2
If node == null, return null immediately (base case).
3
If visited contains the node, return visited.get(node) to avoid infinite cycles.
4
Create cloneNode = new Node(node.val) as the deep copy.
5
Add cloneNode to visited map before recursing (prevents re-visiting).
6
Loop over each neighbor in node.neighbors; recursively call cloneGraph(neighbor) and add the result to cloneNode.neighbors.
7
Return cloneNode.
Java Solution
// class-level visited map
class Solution {
  private Map<Node, Node> visited
      = new HashMap<>();

  public Node cloneGraph(Node node) {

    // ① null check
    if (node == null) return node;

    // ② already cloned?
    if (visited.containsKey(node))
      return visited.get(node);

    // ③ create clone & register
    Node cloneNode = new Node(node.val);
    visited.put(node, cloneNode);

    // ④ recurse over neighbors
    for (Node n : node.neighbors)
      cloneNode.neighbors
        .add(cloneGraph(n));

    return cloneNode;
  }
}
Time O(V + E)
Space O(V)
Pattern DFS + Memoisation
Key idea Register clone before recursing
