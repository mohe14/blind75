Solution · Stack + HashMap
1
Build map of closing → opening brackets: )→(, }→{, ]→[. This lets any close bracket instantly look up its expected counterpart.
) ↓ ( closes
} ↓ { closes
] ↓ [ closes
2
Iterate characters: if it's an open bracket (not in map) → stack.push(c). If it's a close bracket → check the top.
Open bracket
Not in map →
stack.push(c)
Close bracket
stack empty OR
top ≠ map.get(c) → false
else stack.pop()
3
Return stack.isEmpty() — if any unmatched openers remain, the string is invalid.
