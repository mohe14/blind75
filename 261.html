Graph Valid Tree

Given n nodes (labeled 0 to n-1) and a list of undirected edges, determine whether the edges form a valid tree.

A valid tree must be fully connected and contain no cycles.

Example: n = 5, edges = [[0,1],[0,2],[0,3],[1,4]] → true
Example: n = 5, edges = [[0,1],[1,2],[2,3],[1,3],[1,4]] → false
A tree with n nodes always has exactly n−1 edges

Solution · DFS + Adjacency List
1
Early exit: if edges.length != n-1 return false — a tree must have exactly n−1 edges.
2
Build adjacency list: init n empty lists, then for each edge add both directions (edge[0]→edge[1] and edge[1]→edge[0]).
3
DFS from node 0: use a HashSet seenSet. If node not visited → add it, then recurse on all neighbors.
4
Validate connectivity: after DFS, return seenSet.size() == n. If all nodes were visited, the graph is connected.
★
Why it works: edge count check catches cycles; full DFS traversal checks connectivity. Both conditions → valid tree.
