Given a string s and a dictionary wordDict, return true if s can be segmented into a space-separated sequence of one or more dictionary words.

Words in the dictionary may be reused.

s = "leetcode"   dict = ["leet","code"]
i: 0 1 2 3 4 5 6 7 8
dp: T F F F T F F F T
    ↑base       ↑"leet"    ↑"code"
dp[i] = can s[0..i) be fully segmented?
Click to reveal solution
Solution · 1D Dynamic Programming
1
Setup: convert wordDict to a HashSet for O(1) lookups. Create boolean[] dp of size n+1. Set dp[0] = true (empty string is always valid).
2
Outer loop i from 1 to n: each position represents the end of a substring we're trying to validate.
3
Inner loop j from 0 to i: check every possible split point. If the condition holds, mark dp[i] = true and break.
dp[j] == true  &&  words.contains(s.substring(j, i))
★
Key insight: dp[j] means the prefix up to j is valid. If s[j..i) is also a dict word, then dp[i] must be true.
Time
O(n²)
Space
O(n)
