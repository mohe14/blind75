Missing Number

Given an array nums containing n distinct numbers in the range [0, n], return the one number missing from the range.

nums = [3, 0, 1]   n = 3
expected: 0+1+2+3 = 6
actual: 3+0+1   = 4
missing = 6 − 4 = 2 ✓
total = n × (n + 1) / 2
Gauss' formula gives the expected sum in O(1)
Click to reveal solution
Solution · Gauss Sum Formula
1
Expected sum: use Gauss' formula total = n * (n+1) / 2 to compute what the full range [0..n] should add up to.
2
Actual sum: iterate through nums with a for-each loop, accumulating all values into numsTotal.
3
Return the gap: total - numsTotal is exactly the missing value — the difference between what should be there and what is.
Worked example — nums = [9,6,4,2,3,5,7,0,1]
n = 9  →  total = 9×10/2 = 45
numsTotal = 37
missing = 45 − 37 = 8 ✓
Alt — XOR approach: XOR all indices and all values together. Every paired number cancels out, leaving only the missing one. Also O(n) time, O(1) space.

O(n)
