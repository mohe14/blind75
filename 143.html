Reorder List

Given the head of a singly linked list, reorder it so nodes alternate from the front and back: L0 → Ln → L1 → Ln-1 → L2 → …

Do it in-place without modifying node values.

Input:   1 → 2 → 3 → 4 → 5
Output: 1 → 5 → 2 → 4 → 3
↑first                 ↑second (reversed half)
3 phases: find middle → reverse second half → merge
Click to reveal solution
Solution · Find Mid + Reverse + Merge
① Find Middle
② Reverse Half
③ Merge
1
Find middle with slow/fast pointers: slow moves 1 step, fast moves 2. When fast hits null, slow is at the midpoint.
2
Reverse second half starting from slow: use prev = null, current = slow. Standard in-place reversal → prev becomes the new head of the reversed half.
3
Merge the two halves: first = head, second = prev. Interleave until second.next == null.
1 → 5 → 2 → 4 → 3
save first.next → link first→second → link second→saved → advance both
★
Stop condition: while(second.next != null) — the reversed half is always ≤ the first half in length, so it's the safe bound.

O(n)
