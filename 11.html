Container With Most Water

Given an integer array height of length n, find two lines that together with the x-axis form a container that holds the most water.

Return the maximum area of water the container can store.

height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
L↑ ↑R
area = (R − L) × min(height[L], height[R])
Moving the shorter pointer inward is always optimal
Click to reveal solution
Solution · Two Pointers
1
Init pointers: left = 0, right = height.length - 1, maxArea = 0.
2
Compute current area each iteration:
(right − left) × Math.min(height[left], height[right])
3
Move the shorter pointer: if height[left] < height[right] → left++, else → right--. This maximises the chance of finding a taller wall.
4
Track max: maxArea = Math.max(maxArea, currentArea) after each move.
★
Why greedy works: keeping the taller wall and shrinking width can only help — discarding the shorter wall never loses the true maximum.
